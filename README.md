## Getting Started

Primeiro, rode o projeto da API:

[Pokemon API](https://github.com/PedroJardel/pokemon-API)

Depois, rode o projeto Next
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra [http://localhost:3001](http://localhost:3001) com o seu navegador para visualizar o projeto.
